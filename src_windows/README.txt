# Example

> 只能处理24bpp未压缩的bmp


## 编译
运行：c_compile.bat

## 清理
运行：clean.bat

## 使用
1K：960*540
4K：3840*2160
命令行下输入如下命令：
upscale_example.exe <filename> <ouput_width> <output_height> [<output_filename>]
read_header.exe <filename>
downscale.exe <filename> <ouput_width> <output_height> [<output_filename>]

## 算分(4K图像作比较)
使用了绝对路径，test_bmp_4K文件夹不要删除！
